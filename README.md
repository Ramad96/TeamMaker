# Court Rotator

A web application for managing sports court rotations and tournaments. Supports both singles and doubles matches with rotation and tournament modes.

## Features

- **Match Types**: Singles (1v1) or Doubles (2v2)
- **Game Modes**: 
  - Rotation Mode: Continuous play with player rotation
  - Tournament Mode: Bracket-style tournament with finals
- **Score Tracking**: Record match scores (max 21 points)
- **Team Management**: Add/remove players, edit teams during matches
- **Match History**: View past match results

## How to Use

1. Select match type (Singles/Doubles) and game mode (Rotation/Tournament)
2. Add players to the roster
3. Click "Start Session" to begin
4. Tap on winning teams to record scores
5. In Rotation Mode, scores are saved and players rotate automatically

## Deployment

This project is deployed on GitHub Pages. The main entry point is `index.html`.
